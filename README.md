# Knuddels-AppletInfo
Informations about the Applet.

Following informations will be extracted:
- Class of `GroupChat`
- Complete Protocol Tree (for the `GenericProtocol`)
- Version
- Build Date

# Usage
`java -jar AppletInfo.jar <PathToMain.jar>`

# Example
`java -jar AppletInfo.jar C:/test/main.jar`
